# University-search-platform
专为高校打造的微信寻物小程序，是校园里的失物守护站。当你不慎丢失钥匙、证件，或是书本、衣物，都能在此发布寻物启事，附上详细描述与清晰照片。拾到者发现物品后，可第一时间在平台登记。借助智能分类与搜索功能，寻物启事和招领信息能迅速匹配。为保障隐私，平台采用匿名沟通机制，只有双方达成一致，才会交换联系方式，帮助失主和拾物者高效对接，让失物快速“回家” 。
